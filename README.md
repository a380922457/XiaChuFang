# 仿下厨房 iOS APP

本项目采用Swift5仿写，Xcode11环境下开发，采用MVC设计模式，Moya+SwiftyJSON+HandyJSON网络框架和数据解析。  
数据来源charles抓包及爬虫加html解析，如有侵权，请联系我删除
